ReadME. 

* To use this tool, as placed inside this repository, you will have to follow these instructions. 

1. Install, JDK into you computer, saving it in your computer, and installing it in settings.

2. Google "netbeans", download and run the IDE into your computer. 

3. Import the code from GITHUB, by downloading the folder. 

4. Import the code, by creating a new project file inside netbeans.

5. Run the application, depending on the version of netbean installed, prefereably, .11 or .12. 

6. Import the test data into the application. 

7. Check for the results in the desktop application.
